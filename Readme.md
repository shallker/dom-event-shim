
# ie-event-polyfill

  Polyfill W3C DOM event modal on IE8+

## Installation

  Install with [component(1)](http://component.io):

    $ component install shallker/ie-event-polyfill

## Embed in HTML
```html
<!--[if IE 8]>
  <script src="http://shallker.github.io/ie-event-polyfill/index.js">
<![endif]-->
```
