
# dom-event-shim

  Polyfill W3C DOM event modal on IE8+

## Installation

  Install with [component(1)](http://component.io):

    $ component install shallker/dom-event-shim

## Embed in HTML
```html
<!--[if IE 8]>
  <script src="http://shallker.github.io/dom-event-shim/index.js"></script>
<![endif]-->
```
